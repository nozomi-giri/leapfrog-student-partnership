# LSPP Assignment

A React + TypeScript recreation of the Leapfrog Student Partnership Program landing page, built for the LSPP assignment after the React workshop.

🔗 Live Demo: https://leapfrog-student-partnership.vercel.app/

## Features

- Dark mode toggle
- Go to top button
- Image gallery with carousel
- Animated stats counters
- FAQ accordion
- Mentors section
- Responsive design

## Running locally

bash
git clone https://github.com/nozomi-giri/leapfrog-student-partnership.git
cd leapfrog-student-partnership
npm install
npm run dev
